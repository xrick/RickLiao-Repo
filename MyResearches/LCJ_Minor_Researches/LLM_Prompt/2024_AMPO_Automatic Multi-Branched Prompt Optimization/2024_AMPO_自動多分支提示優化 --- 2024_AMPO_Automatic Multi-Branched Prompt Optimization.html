Algorithm 1 AMPO
Require: Initial prompt: \(P_{0}\), Training set: \(D_{\text {train }}\), Valida-
    tion set: \(D_{\text {val }}\), Iteration limit: \(T\)
    // LLM agents
    Analyzer: Conduct error analysis.
    Summarizer: Condense error reasons into patterns and
    assign important scores.
    Revisor: Edit the multi-branched prompt.
    Initiate \(P=P_{0}\)
    while \(t \leq T\) do
        Evaluate \(P\) on \(D_{\text {train }}\) and sample \(K\) failed cases
    \(E=\left\{\left(x_{i}, y_{i}\right) \mid\left(x_{i}, y_{i}\right) \in D_{\text {train }} \wedge L L M\left(x_{i} ; p\right) \neq y_{i}\right\}\)
        // Pattern Recognition
        LLM-Analyzer conducts error analysis and iden-
    tifies the error reasons for each failed case: \(R=\)
    \(\left\{r_{1}, r_{2}, \ldots r_{K}\right\}\)
        LLM-Summarizer condenses \(R\) reasons into \(M\) pat-
    terns and assign important scores from each pattern:
    Patt \(=\left\{\left(\right.\right.\) patt \(\left._{1}, s_{1}\right), \ldots\left(\right.\) patt \(\left.\left._{M}, s_{M}\right)\right\}\)
        // Branch Adjustment
        Selects top \(N\) patterns by important scores.
        LLM-Revisor optimizes \(P\) based on top \(N\) Patt and
    obtained optimized prompts \(P_{\text {opt }}=\left\{P 1, \ldots P_{N}\right\}\).
        // Branch Pruning
        LLM-Revisor prunes the optimized prompts \(P_{o p t}\) and
    obtained \(P_{\text {pruned }}=\left\{P 1, \ldots P_{N}\right\}\)
        Evaluate \(N\) pruned prompts \(P_{\text {pruned }}=\)
    \(\left\{P 1, \ldots P_{N}\right\}\) on \(D_{\text {val }}\) and select the best prompt
    \(P^{*}\).
        Update \(P=P^{*}\)
    end while
    return The best prompt.
